# Sloth Documentation

## Setup venv

Create venv

```shell
python3 -m venv .venv
```

Activate venv

```shell
source .venv/bin/activate
```

Install required packages:

```shell
(.venv) pip install -r requirements
```

Serve the doc locally:

```shell
(.venv) mkdocs serve
```

Shutdown venv

```shell
(.venv) deactivate
```
